# Lab Work: Light Bulb Finite Automaton 

## Цель работы
Реализовать конечный автомат для управления состояниями лампочки с визуализацией в WPF.  
Показать работу состояний, переходов и таймеров в простом интерфейсе.

## Описание автомата

- Автомат моделирует три состояния лампочки:

  1. **Off** — лампочка выключена  
  2. **On** — лампочка включена  
  3. **BurnedOut** — лампочка перегорела  

- **Начальное состояние:** Off  
- **Перегорение:** лампочка автоматически переходит в BurnedOut через 5 секунд работы  

## Переходы между состояниями

- **Off → On:** пользователь нажимает кнопку "Переключить"  
- **On → Off:** пользователь нажимает кнопку "Переключить"  
- **On → BurnedOut:** автоматически через 5 секунд работы (таймер)  
- **BurnedOut → Off:** пользователь нажимает кнопку "Заменить лампочку"  

## Реализация

- **WPF интерфейс** с элементами:
  - `LightBulb` — визуальное представление лампочки  
  - `ToggleButton` — кнопка включения/выключения  
  - `ReplaceButton` — кнопка замены лампочки  
  - `StatusText` и `TimerText` — отображение состояния и времени  

- **DispatcherTimer** с интервалом 1 секунда обеспечивает отслеживание времени работы лампочки и переход в BurnedOut.  
- **UpdateUI()** обновляет интерфейс в зависимости от состояния автомата.  

## Особенности

- Таймер сбрасывается при выключении или включении лампочки  
- Кнопки активируются/деактивируются в зависимости от состояния лампочки  
- Цвет лампочки отражает текущее состояние:  
  - Серый — Off  
  - Жёлтый — On  
  - Тёмно-красный — BurnedOut  

## Результат работы

- Лампочка корректно переключается между состояниями  
- Автомат демонстрирует работу конечного автомата с визуальной обратной связью  
- Пользователь может наблюдать таймер работы и замену лампочки после перегорания
