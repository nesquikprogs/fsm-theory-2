# Lab Work: Text Generation with Markov Chains

## Цель работы
Реализовать генератор текста на основе **цепей Маркова**.  
Программа анализирует исходный текст, строит статистику переходов между словами и позволяет генерировать новый текст в стиле исходного материала.

## Принцип работы

- **Цепь Маркова**: математическая модель, где вероятность перехода в следующее состояние зависит только от текущего состояния, а не от всей истории.  
  В текстовом контексте: каждое следующее слово выбирается только на основе текущего слова.

- **Матрица переходов**:  
  Для каждого слова хранится информация о том, какие слова чаще всего следуют за ним и с какой вероятностью.

- **Генерация текста**:  
  - Пользователь выбирает начальное слово.  
  - Новый текст формируется пошагово или автоматически.  
  - Каждое следующее слово выбирается случайным образом с учётом вероятностей переходов, что позволяет имитировать стиль исходного текста.

## Возможности программы

- Пошаговая генерация текста  
- Автоматическая генерация заданной длины  
- Имитация стилистики исходного текста  
- Возможность анализа любого текстового материала  

## Особенности реализации

- Использование словаря или хеш-таблицы для хранения переходов  
- Случайный выбор следующего слова на основе вероятностей  
- Простая масштабируемая архитектура для добавления новых функций (например, генерация по N-граммам)
