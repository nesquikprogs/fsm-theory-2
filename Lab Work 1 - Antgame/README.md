# Lab Work: Ant Game - DFA Simulation

## Цель работы
Разработка простой симуляции поведения муравья с использованием **конечного детерминированного автомата (DFA)**.  
Муравей имеет три состояния и переходит между ними в зависимости от окружающей среды:

1. **Fleeing** — убегание от курсора мыши  
2. **Collecting** — сбор листика  
3. **ReturningHome** — возврат в норку с листиком  

## Описание работы программы

1. При запуске создается игровое поле размером 800x600 пикселей, норка муравья находится в центре.  
2. Муравей начинает в состоянии **Collecting** и ищет листик.  
3. Если курсор приближается к муравью ближе, чем заданное расстояние страха (`FearDistance`), состояние меняется на **Fleeing** — муравей убегает от курсора.  
4. После сбора листика муравей переходит в состояние **ReturningHome** и несет листик в норку.  
5. По достижении норки листик засчитывается, и DFA возвращается в состояние **Collecting** для поиска нового листика.  
6. Программа визуализирует:  
   - Муравья (красный квадрат)  
   - Листики (зеленые квадраты)  
   - Норку (синий квадрат)  
   - Текущие состояния DFA и счет собранных листиков  

## Управление

- Перемещайте курсор мыши, чтобы проверять реакцию муравья (избегание курсора).  
- Муравей автоматически собирает листики и возвращается домой.  

## Особенности реализации DFA

- Состояния муравья описаны через `enum AntState`  
- Переходы между состояниями реализованы через `switch` в методе `Update()`  
- Проверка границ игрового поля (чтобы муравей не вышел за пределы)  
- Движение реализовано через нормализацию векторов для плавной анимации  

## Результат работы

При успешном выполнении программы:  
- Муравей собирает листики  
- Реагирует на приближение курсора  
- Счет собранных листиков увеличивается на экране  
- Визуально отображается состояние DFA
