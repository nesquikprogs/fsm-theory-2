# Lab Work: Traffic Light Hierarchical Finite State Machine (FSM)

## Цель работы
Реализовать иерархический конечный автомат (FSM) для управления светофором с визуализацией в WPF.  
Демонстрировать работу подсостояний и управление переходами с помощью таймера.

## Описание автомата

- Автомат управляет светофором с тремя основными состояниями:
  1. **Off** — светофор выключен  
  2. **Operating** — нормальная работа  
  3. **Emergency** — аварийное состояние  

- В состоянии **Operating** реализуется последовательность подсостояний (циклически):
  1. **Red** — красный сигнал  
  2. **RedYellow** — красный + жёлтый  
  3. **Green** — зелёный  
  4. **Yellow** — жёлтый  

- **Начальное состояние:** Off  

## Переходы между состояниями

- **Off → Operating:** включение светофора  
- **Operating → Off:** выключение светофора  
- **Operating → Emergency:** аварийная ситуация (может быть задана вручную)  
- **Emergency → Operating:** восстановление работы  

- Внутри **Operating** переходы между подсостояниями управляются таймером:
  - Каждую секунду счётчик уменьшается и вызывается переход к следующему подсостоянию  
  - Цикл: Red → RedYellow → Green → Yellow → Red  

## Реализация

- **WPF интерфейс** с элементами:
  - Отображение цветов светофора (Red, Yellow, Green)  
  - Кнопки управления: включение/выключение, аварийный режим  
  - Таймер DispatcherTimer для управления переходами  

- **Иерархический FSM**:
  - Основные состояния: Off, Operating, Emergency  
  - Подсостояния внутри Operating: Red, RedYellow, Green, Yellow  
  - Управление состояниями реализовано через методы переходов и обновление UI  

## Особенности

- Автомат демонстрирует работу иерархического конечного автомата с подсостояниями  
- Таймер обеспечивает автоматический переход между подсостояниями  
- Цвета светофора обновляются в UI в реальном времени  

## Результат работы

- Светофор корректно циклирует через подсостояния в рабочем режиме  
- Возможность переключения в Off и Emergency режимы  
- Пользователь наблюдает визуальную работу FSM и управление переходами
