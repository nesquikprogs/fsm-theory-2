# Лабораторные работы по Теории автоматов

В этом репозитории собраны лабораторные работы по дисциплине **Теория автоматов**.  
Работы охватывают различные виды автоматов и клеточных систем, включая сети Петри, конечные автоматы, клеточные автоматы и цепи Маркова.  

**Используемые технологии:**  
- C# (Windows Forms) — для реализации алгоритмов и визуализации автоматов  
- HTML / JavaScript — для интерактивной визуализации и демонстрации работы автоматов в браузере  
- Общие алгоритмические методы для моделирования автоматов и анализа цепей  

---

## Особенности

- Все лабораторные работы реализованы как **интерактивные симуляции**, позволяющие наблюдать работу автоматов в реальном времени.  
- Использование **C# и Windows Forms** обеспечивает удобный интерфейс для пошаговой визуализации.  
- Веб-версии на **HTML и JavaScript** позволяют запускать модели в браузере без установки дополнительного ПО.  
- Репозиторий демонстрирует практическое применение алгоритмов и моделей из курса «Теория автоматов» для изучения поведения систем, обработки данных и генерации последовательностей.
